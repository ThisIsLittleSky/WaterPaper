# 课程论文结构模板

本科课程期末论文（3000-8000 字）的章节结构模板，按学科类型分类。

## 通用结构（适用于大部分学科）

```
摘要 + 关键词
一、引言
二、[核心论述章 1]
三、[核心论述章 2]
四、结论
参考文献
```

字数分配（以 5000 字为例）：

| 章节 | 字数 | 占比 |
|------|------|------|
| 摘要 | 200-300 | 5% |
| 引言 | 800-1000 | 18% |
| 正文章 1 | 1500-1800 | 32% |
| 正文章 2 | 1500-1800 | 32% |
| 结论 | 400-600 | 10% |
| 参考文献 | — | 3%（不计入正文） |

## 按学科分类

### 1. 经管类（管理学、经济学、市场营销）

```
摘要 + 关键词
一、引言
  1.1 研究背景与意义
  1.2 国内外研究现状
  1.3 研究方法
二、理论基础与现状分析
  2.1 核心概念界定
  2.2 相关理论基础
  2.3 现状分析
三、问题分析与对策/实证分析
  3.1 存在的问题
  3.2 原因分析
  3.3 对策建议 / 实证结果
四、结论
参考文献
```

### 2. 人文社科类（文学、历史、哲学、社会学）

```
摘要 + 关键词
一、引言
  1.1 选题背景
  1.2 研究意义
  1.3 文献综述
二、[核心分析章 1：理论/背景分析]
三、[核心分析章 2：案例/文本/现象分析]
四、结论
参考文献
```

### 3. 理工类（计算机、电子、机械）

```
摘要 + 关键词
一、引言
  1.1 研究背景
  1.2 国内外研究现状
  1.3 本文工作
二、[技术/方法 1]
三、[技术/方法 2 或 实验/应用分析]
四、结论
参考文献
```

### 4. 案例分析类（法学、医学、教育学）

```
摘要 + 关键词
一、引言
  1.1 研究背景
  1.2 研究目的与意义
二、案例介绍
  2.1 案例概况
  2.2 案例选取依据
三、案例分析
  3.1 问题识别
  3.2 原因探析
  3.3 启示与建议
四、结论
参考文献
```

## 章节命名规范

- 一级标题用"一、二、三、四"，不用"第X章"
- 二级标题用"1.1 1.2"，不用"（一）"
- 标题不超过 15 个字
- 标题中不出现标点符号（问号、感叹号除外）

## 参考文献数量

| 论文字数 | 中文文献 | 英文文献 | 总计 |
|----------|----------|----------|------|
| 3000-4000 | 4-6 | 1-2 | 5-8 |
| 4000-6000 | 5-7 | 2-3 | 7-10 |
| 6000-8000 | 6-8 | 2-4 | 8-12 |

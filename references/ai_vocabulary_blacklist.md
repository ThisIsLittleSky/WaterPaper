# 中文学术 AI 高频词汇风险参考

> **来源**: 整合自 thesis-optimizer 同类文档 + WaterPaper 原有黑名单
> **用途**: 写作时规避；`humanize_check.py` 检测词库；改写时参照替换

本表列出中文学位/课程论文中常见的模板化词汇和空泛表达。使用时应结合上下文判断，优先保留术语准确性和作者原意。

---

## 分级体系

| 等级 | 含义 | 处理策略 |
|------|------|---------|
| 🔴 **红色（高）** | AI 生成文本中高频出现。无具体信息时应删除、压缩或替换。 | 优先消灭 |
| 🟡 **黄色（中）** | 人类也使用，但密度过高时模板化明显。 | 控制密度，分散或替换 |
| 🟢 **绿色（低）** | 本身正常，但所在段落若已有红/黄词密集时需留意。 | 正常使用，关注密度 |

## 🔴 高风险词

### 累赘的连接与过渡词

| AI 高频词 | 处理策略 |
|-----------|---------|
| **此外** / 更有甚者 / 与此同时 | 删除或用分号连接；改用"另外""同时" |
| **在这一背景下** / 在这个瞬息万变的时代 | 结合具体场景重写，或直接删除 |
| **综上所述** / 总而言之 / 概括来说 | 改用"本节表明""实验结果表明""由此可见" |

### 夸大的修饰与评价词

| AI 高频词 | 处理策略 |
|-----------|---------|
| **至关重要** / 不可或缺 / 极其重要 | 改用"关键""核心""对...影响显著" |
| **深远影响** / 革命性突破 / 完美 | 改用"显著提升""有效改善""取得了统计学意义的改进" |
| 卓越的 / **宝贵的经验** / 充满活力的 | 改用"优异的""可靠的基准""客观证据" |

### 空泛的动作描述词

| AI 高频词 | 处理策略 |
|-----------|---------|
| **深入探讨** / 全面分析 / 系统梳理 | 改用"详细评估""重点考察""针对...进行了推导" |
| **充分体现** / 淋漓尽致地展现 | 改用"表明""证实""符合...的假设" |

### WaterPaper 原有补充

| 禁用表述 | 处理策略 |
|-----------|---------|
| 首先/其次/再次/最后 | 自然过渡，不用序数词 |
| 具有重要的理论意义和现实意义 | 具体说明为何重要 |
| 随着……的快速发展 / 在当今时代背景下 | 直接陈述问题 |
| 为……奠定基础 / 在……的过程中 | 改为具体描述 |
| 实现了良好效果 / 具有较高价值 | 用数据或具体分析替代 |
| ———— / ———— 长横线 | 用空白行或章节标题替代 |

## 🟡 中风险词

### 结构与逻辑连接

- **然而 / 但是** → 替代：虽然 / 尽管如此 / 反观 / 与之相对
- **因此 / 鉴于此 / 基于此** → 替代：于是 / 从而 / 使得 / 故本文...
- **值得注意的是 / 有必要指出** → 替代：更为特殊的是 / 一个值得关注的现象是 / 直接说重点

### 学术操作动词

- **揭示了 / 阐明了** → 替代：指出 / 发现了 / 说明 / 解释了
- **构建 / 打造 / 提供** → 替代：设计 / 提出 / 实现 / 搭建
- **缓解 / 解决** → 替代：改善 / 克服 / 减轻

### 名词化表达

- **格局 / 全景 / 景象** → 替代：现状 / 分布 / 情况
- **织锦 / 交织 / 融合** → 替代：结合 / 组合 / 交叉应用

## 🟢 低风险词

正常使用即可，只在同段红/黄词密集时关注：

- **表示包含/涉及**：涵盖、涉及、囊括、包含、具备
- **表示探究/分析**：剖析、摸索、探寻、探索、评估
- **表示结果/影响**：促使、推动、赋能、催生、导致

## 实战词汇重构法则

### 法则 1：去名词化 (De-nominalization)

AI 喜欢把动词名词化以显得高级。将抽象名词还原为具体及物动词。

> ❌ "特征解耦机制起到了**至关重要**的作用，**提供了**解决过拟合的**新思路**。"
>
> ✅ "特征解耦机制有效**抑制**了过拟合，**避免**了特征塌陷。"

### 法则 2：废词剥离 (Cut the fluff)

AI 句子开头常挂废话。直接从主语切入。

> ❌ "**值得注意的是**，上述实验结果**充分证明了**本文模型架构的优越性。"
>
> ✅ "上述实验结果验证了本文模型的有效性。"

### 法则 3：具体化替换

优先使用论文中已有的具体对象、指标、实验设置替代空泛套话。

- 不为了"不像 AI"引入生僻词、古风表达或口语化转折
- 需要连接关系时，选符合中文学术语体的平实表达

## 自查清单

改写后逐段检查：
- [ ] "此外、至关重要、探讨" 等红词是否密集或空泛
- [ ] "然而"、"因此" 等连接词是否连续重复
- [ ] 所有评价性质的主观溢美词是否已移除
- [ ] 段落开头前 4 个字能否删除而不改变意思（若能，多半是 AI 过渡词）
- [ ] 无长横线分隔符

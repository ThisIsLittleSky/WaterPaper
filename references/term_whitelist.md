# 术语保护白名单

> **来源**: 提取自 thesis-optimizer 策略A技术7"术语保护机制"
> **用途**: 降AI/降重改写时的安全边界——以下内容**禁止修改**

降AI和降重改写时，最大的风险不是改得不够，而是**误改了正确的专业表述**，造成事实错误或术语不一致。

---

## 保护清单

### 1. 数学公式与变量

即使它们导致局部查重升高或句长指标不达标，也**保持不变**。

```
MSE, F1-score, σ, θ, α, β, O(n²), p-value 等
```

### 2. 专业术语与英文缩写

**禁止**为了降重或降AI把专业术语"通俗化"：

| 正确（保留） | 错误（禁止） |
|-------------|-------------|
| Transformer | "变换器模型" |
| CNN | "卷积神经网络"（如原文用CNN就保持CNN） |
| Self-Attention | 不能改成"自我关注" |
| 时间序列预测 | 不能改成"时序推测" |
| 知识蒸馏 | 不能改成"知识浓缩" |

### 3. 模型/算法/系统名称

```
BERT, GPT, ResNet, LSTM, Autoformer, Informer, 
YOLO, U-Net, Adam, Softmax, Backpropagation 等
```

### 4. 参考文献与引用作者

- `[1]` `[2-4]` 等引用编号 → 保持不动
- 作者姓名 → 保持不动
- 论文标题（在参考文献列表中）→ 保持不动

### 5. 专有名词

- 人名、地名、机构名
- 数据集名称（ImageNet, CIFAR-10, SQuAD 等）
- 特定领域术语（如法律条文编号、医学术语、化学式）

---

## 执行准则

在应用任何降AI/降重改写时：

1. **遇到白名单内容 → 绕过**，不修改该词本身
2. **只调整包围白名单内容的外围结构**：连词、动词、状语、语序
3. **如果一段中白名单内容占比超过 40%**：该段只做最小干预（连接词替换 + 句长微调），不做结构重组

### 示例

> ❌ 错误改写（改了术语）：
> 原文 "Transformer 通过 Self-Attention 实现全局建模"
> 改写 "变换器模型借助自我关注机制达成整体特征提取"

> ✅ 正确改写（保护术语，只调外围）：
> 原文 "Transformer 通过 Self-Attention 实现全局建模"
> 改写 "Transformer 架构借助 Self-Attention 组件捕捉序列中的全局依赖关系"

---

## 与黑名单的关系

| | 黑名单 (`ai_vocabulary_blacklist.md`) | 白名单 (本文档) |
|---|---|---|
| 作用 | 标记需删除/替换的词汇 | 标记禁止修改的词汇 |
| 优先级 | 低（先看白名单） | **高（优先判断）** |
| 冲突时 | 让步 | **优先** |

**原则**：宁可保留一个黄词（如术语周围的"然而"），也不能误改白名单中的专业术语。

# 默认格式规范

本科课程期末论文（3000-8000 字）标准格式，基于 GB/T 7714 和常见中文高校课程论文要求。

## 页面设置

| 属性 | 值 |
|------|-----|
| 纸张 | A4 |
| 上边距 | 2.54 cm |
| 下边距 | 2.54 cm |
| 左边距 | 3.18 cm |
| 右边距 | 3.18 cm |
| 页眉 | 1.5 cm |
| 页脚 | 1.75 cm |

## 标题格式

| 层级 | 字体 | 字号 | 加粗 | 对齐 | 段前 | 段后 | 行距 |
|------|------|------|------|------|------|------|------|
| 论文标题 | 黑体 | 二号 | 加粗 | 居中 | 0 | 0 | 1.5 |
| 一级标题 | 黑体 | 小三 | 加粗 | 左对齐 | 12pt | 6pt | 1.5 |
| 二级标题 | 黑体 | 四号 | 加粗 | 左对齐 | 6pt | 3pt | 1.5 |
| 三级标题 | 黑体 | 小四 | 加粗 | 左对齐 | 3pt | 0 | 1.5 |

## 正文格式

| 属性 | 值 |
|------|-----|
| 中文字体 | 宋体 |
| 英文/数字字体 | Times New Roman |
| 字号 | 小四（12pt） |
| 行距 | 1.5 倍 |
| 首行缩进 | 2 字符 |
| 段前 | 0 |
| 段后 | 0 |

## 摘要格式

| 属性 | 值 |
|------|-----|
| "摘要"标题 | 黑体小四加粗居中 |
| 摘要正文 | 宋体小四，首行缩进 2 字符 |
| 关键词标签 | 黑体小四加粗，顶格 |
| 关键词内容 | 宋体小四 |
| 关键词数量 | 3-5 个 |
| 关键词分隔 | 分号（；） |

## 图表格式

| 属性 | 值 |
|------|-----|
| 图题 | 宋体五号居中，置于图下方 |
| 表题 | 宋体五号加粗居中，置于表上方 |
| 图表内容字号 | 宋体五号 |
| 图表编号 | 按章节或全文连续编号 |

## 参考文献格式（GB/T 7714）

### 期刊论文
```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```

### 专著
```
[序号] 作者. 书名[M]. 出版地: 出版社, 年.
```

### 学位论文
```
[序号] 作者. 题名[D]. 保存地点: 保存单位, 年.
```

### 参考文献列表格式
- 宋体五号
- 悬挂缩进 0.74 cm
- 编号用方括号 [1] [2] ...

## 示例

```
[1] 张三, 李四. 数字化转型对企业绩效的影响研究[J]. 管理世界, 2022, 38(5): 123-135.
[2] Smith J, Brown K. Artificial Intelligence in Business[J]. Harvard Business Review, 2021, 99(3): 45-52.
```
